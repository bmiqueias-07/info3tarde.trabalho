# info3tarde.trabalho

Meu projeto funcionará como uma plataforma para gerenciar projetos, semelhante a um formulário interativo. Nele, será possível cadastrar e remover usuários interessados em participar da Mansão onde o nome é **Sibaúma Gamer** contendo um sistema que permitirá o registro de informações como: nome completo, idade, jogos de interesses e tipos de apostas, atividades e até fazer lives como um stream no youtube ou outras plataforma desejada. As atividades poderão incluir opções como trabalhos em grupo, partidas de jogos, interclasse, confrontos x1, x2, entre outros formatos com diferentes números de participantes. Exemplo de jogos disponíveis estarão títulos populares como Roblox, Free Fire, Fortnite, além de atividades como futebol e outras competições. O objetivo é organizar e facilitar a participação dos usuários em eventos e desafios dentro da Mansão Gamer, criando um ambiente dinâmico, competitivo, amigavel e divertido.
