
# Seleção de Candidatos

**COMO** administrador da Mansão _Sibaúma Gamer_  
**QUERO** seleciona novos candidatos interessados em participar das atividades da plataforma  
**PARA** que eu possa organizar os eventos, jogos e atividades disponíveis para o candidato.  

**Critérios de Aceitação:**

- Deve ser possível informar o nome do usuário  
- Deve ser possível informar a data de nascimento, quantos anos e interesses  
- O sistema deve impedir o cadastro de pessoas com nomes duplicados, a não ser que prove ser irmão com o mesmo nome e ser da mesma família  
- O sistema deve selecionar candidatos com ficha tecníca organizada
