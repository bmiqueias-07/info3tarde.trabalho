
# Cadastro de candidatos

**COMO** um interessado em participar a _Simbauma Gamer_  
**QUERO** me cadastrar como candidato da mansão   
**PARA** que eu possa acompanhar o processo seletivo.     

**Critérios de Aceitação:**

- A tela deve apresentar um formulário de cadastro   
- Apenas maiores de idade de 15 anos para cima poderão submeter o seu interesse 
- O sistema deve validar a idade informada pelo candidato.
- O candidato deve receber uma confirmação após realizar o cadastro.
- A candidatura deve possuir um status inicial de "Em análise".
- Os dados cadastrados devem ser armazenados para consulta futura.
